# Contributing

Padrões rápidos para contribuir com os learning-paths.

- Branches: `feat/<tema>`, `fix/<tema>`, `docs/<tema>`.
- Commits: verbo no imperativo (ex.: "Add exercícios Nível 1 de lógica").
- PRs: pequenas e focadas; peça revisão por pelo menos 1 colega.
- Testes: inclua testes simples para exercícios e rode `npm test` onde aplicável.

Processo
1. Crie uma branch `feat/...` e faça commits pequenos.
2. Abra PR com descrição curta, checklist e impacto em outras trilhas.
3. Após revisão e aprovação, faça merge.
