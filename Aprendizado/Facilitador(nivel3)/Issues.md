# Issue 

Uma issue é um REGISTRO de algo que precisa ser feito ou resolvido dentro de um projeto.

Pode ser:

um problema

uma tarefa

uma ideia

uma melhoria

um bug

Tudo anotado oficialmente no GitHub, ligado ao repositório.

Ao clicar em new Issue, e abrir uma, uma Issue ja foi feita com determinada funcionalidade.
