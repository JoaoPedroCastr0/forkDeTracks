## III
- Git status: Comando responsável por dizer quais arquivos foram ALTERADOS e posteriormente prontos para realização de um commit, sinalizando também quais arquivos NÃO FORAM RASTREADOS.

## Configurações de SSH

Foram utilizados os seguintes comandos para utilização de chave SSH : 

- ssh-keygen -t ed25519 -C "seu@email.com"
- Iniciar agente: eval "$(ssh-agent -s)"
- sua chave: ssh-add ~/.ssh/id_ed25519
- Copia pública: cat ~/.ssh/id_ed25519.pub
